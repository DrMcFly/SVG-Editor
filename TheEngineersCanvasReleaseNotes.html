The Engineers Canvas Release Notes

1.0 (12/22/2023)

A simple web application where you can draw a line, a circle or a rectangle - but only one instance per shape (so far…).

1.0.1 (12/24/2023)

  • Made the coordinate hover box hideable.
  • Added support for rectangles to be created with negative width/height.
  • Added the ability to click out of the context menu.

        1.0.2 (12/24/2023)

  • Users can finally add multiple instances of a shape.
  • Finalized the SVG export function.
  • Added custom naming function.
  • Implemented custom incrementing IDs for shapes
  • Added construction lines/shapes.
  • Added a clear button
